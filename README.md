Running the Project
1. Clone the repository:
git clone https://github.com/<your-username>/<repo-name>.git
cd <repo-name>
2. Setup backend:
cd backend
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
3. Ensure MongoDB is running
4. Start server:
python app.py

Visit:
http://127.0.0.1:5000
